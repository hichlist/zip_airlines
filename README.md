# Zip Airlines

This is Zip Airlines's API for managing airplanes, to get information and add new airplanes.

## Requirements

- python 3.6
- django 3.2
- rest_framework
- markdown 3.3.7 

## Installation

- python3 manage.py migrate
- python3 manage.py createsuperuser

## Start
- python3 manage.py runserver
